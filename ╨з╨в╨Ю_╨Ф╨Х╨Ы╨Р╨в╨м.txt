ОБНОВЛЕНИЕ (усиленный будильник)

1. Распакуй этот архив.
2. На GitHub в репозитории obeya-app: Add file -> Upload files.
3. Перетащи в окно папку "app" из этого архива.
   Это ПЕРЕЗАПИШЕТ 7 изменённых файлов, остальное не тронет.
4. Внизу -> Commit changes.
5. Сборка запустится сама (вкладка Actions). Жди зелёную галочку ~5 мин.
6. Скачай новый app-debug.apk (Artifacts внизу сборки).
7. На телефоне просто установи поверх старого (данные и филиал сохранятся).
8. Открой приложение один раз -> в шторке появится значок "Obeya задачи".
   Это и есть служба, которая держит будильник. Так и должно быть.
